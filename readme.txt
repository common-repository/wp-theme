== Description ==

WP Themes Plugin allow you to preview a theme without activating it from admin panel.
This plugin is great for sites that offers wordpress themes but can be used by anyone need to see how a theme looks on his site without making it visible to everyone who visits the site.
WP Themes Plugin can help you to check if a theme have errors before activating it for all visitors.
Also with this plugin you can show your visitors how your website looks with other themes and maybe leave then to vote what theme they like.

You can get some free [wordpress themes](http://extremedata.eu) from my website extremedata.eu
[DotEU Blue](http://extremedata.eu/projects/doteu-blue/) is my first [free wordpress theme](http://extremedata.eu/projects) available [here on wordpress.org](http://wordpress.org/extend/themes/doteu-blue) and on my website.

== Installation ==

1. Upload the `wp_themes` folder and its contents to your `wp-contents/plugins` directory.
2. Activate the plugin from the `Plugins` menu.
3. Go to WpThemes options page (from `Settings` menu) and read more instructions there.
